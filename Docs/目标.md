目前opencode的交互中，恢复之前的历史聊天会话非常不方便
只看标题很难确认是不是这个会话

所以，开发一个网页，可以自动找到opencode的历史会话记录
然后用更简洁大气的形式展示出来历史会话的标题和内容
这样，用户就可以通过具体内容来快速找到自己需要的会话

默认展示最近10条会话，点击会话，右侧可以展开查看具体内容

有搜索功能，可以搜索匹配的字符串，按照时间先后倒序排列，最靠近的会话在最上面

进入网页，可以向用户索求权限，用于访问opencode保存历史的特定文件夹，从而获取历史会话记录

历史和下面几个文件或者目录有关，我不确定每个文件的内容，你可以自行查看

- `~/.local/share/opencode/opencode.db`
- `~/.local/share/opencode/storage/session/`
- `~/.local/share/opencode/storage/message/`
- `~/.local/share/opencode/storage/part/`

每次用户查看对应的会话的时候，有一个地方展示当前会话的id，并且可以一键复制打开对应会话的命令
```shell
opencode --session <session_id>
```
